# 📊 Monthly Sales Dashboard (Excel)

## Project Overview

This project is an interactive Monthly Sales Dashboard developed using Microsoft Excel to analyze sales performance and generate meaningful business insights.

The dashboard was created by applying Excel concepts such as Pivot Tables, Pivot Charts, KPI calculations, Data Cleaning, Data Visualization, and Interactive Slicers.

--

## Objectives

* Analyze overall sales performance.
* Track monthly sales trends.
* Monitor shipment and product performance.
* Identify top-performing sales representatives.
* Create an interactive dashboard for business decision-making.
